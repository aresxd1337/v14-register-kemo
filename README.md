# Nasıl kuracağım ?

öncelikle ```config.json'u``` doldur ve sonradan ```npm i``` yaparak modülleri indir sonradan ```start.bat``` basıp botu başlatın. 


# Bilgilendirme
işinize yaradıysa bir start atarsınız artık. :) ❤️
